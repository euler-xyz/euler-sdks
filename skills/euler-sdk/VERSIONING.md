# Skill Versioning

`euler-sdk` uses semantic versioning.

- MAJOR: breaking changes to guidance structure or expected workflow.
- MINOR: new sections/rules or materially expanded guidance.
- PATCH: wording clarifications, typo fixes, non-breaking refinements.

Version must be updated in all of:
- `SKILL.md` frontmatter: `metadata.version`
- `metadata.json`: `version`
- `AGENTS.md` header version
- `CHANGELOG.md`
