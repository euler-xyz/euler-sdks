# Changelog

All notable changes to `euler-v2-sdk` should be documented in this file.

## [0.1.0-beta] - 2026-04-17

### Added

- V3-backed account, vault, EulerEarn, rewards, intrinsic APY, and vault-type adapters for the default SDK data path.
- Securitize collateral vault support, including app details and dedicated example screens.
- Expanded React and script examples with query profiling, query options, raw entity inspection, totals, and swap simulation flows.

### Changed

- Improved oracle decoding, resolved vault metadata, price normalization, reward APY handling, and batched vault/account fetching.
- Updated simulation approval handling, Pyth Hermes requests, state override behavior, and parity test coverage.
